JDBC：java DateBase Connectivity
用来访问数据库的，定义了一套接口

主要接口：
1)DriverManager 驱动管理
2)Connection 连接接口
3)Statement 语句对象   PrepareStatement
4)ResultSet 结果集接口

JDBC工作过程：
Oracle版
1)加载驱动，建立连接
class.forName("oracle.jdbc.driver.OracleDriver");
Connection con = DriverManager.getConnection("oracle:thin:@localhost(127.0.0.1):1521:orcl","user","pwd");
2)创建语句对象
Statement st = con.createStatement();
PreparedStatement ps = con.prepareStatement(sql);
3)执行SQL语句
st.executeQuery(sql);
ps.executeQuery();
4)处理结果集
ResultSet rs = st.executeQuery(sql);
while()	rs.next
5)关闭连接
if(rs != null)
	rs.close();
if(st != null)
	st.close();
if(con != null)
	con.close();
