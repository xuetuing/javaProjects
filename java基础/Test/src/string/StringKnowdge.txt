String相关知识：
String(不可变) 和  用于操作字符串的类 StringBuilder(可变),StringBuffer(不常用)
String为什么不可变？-->private final char value[];

1）普通String : String = char[] + 操作,	长度，内容不可变

常用方法如下：
查找相关：charAt()		indexOf()	lastIndexOf()
大小写：toupperCase()		toLowerCase()	
数组字符串的转换：toCharArray()	toString()
判断相关：equals()		startsWith()	endsWith()	isEmpty()
字符串匹配：matches()	split()
字符串截取：trim()		subString()
获取编码：getBytes()	hashCode()

String-->byte[]		str.getBytes()
byte[]-->String		String str = new String(byte[] b,[int start,int len,]charSet);通过String构造方法转为字符串

String-->char[]		str.toCharArray()
char[]-->String		new String(char[])

2）StringBuilder = char[] + 操作		可变
构建方式：StringBuilder builder = new StringBuilder(str);
其中的数组内容可变，如果长度不够，会利用变长数组算法来维护，自动扩容
StringBuilder是用来修改字符串的类，是一个变长字符序列
StringBuilder     Jdk1.5   	效率较高		线程不安全
StringBuffer	  JDK1.0	效率相对较低	线程安全

常用方法 ：append()、 insert()、 delete()、 replace(),reverse() 返回值都是StringBuilder
对象本身的引用，如果我们的软件需要大量字符串处理，建议使用StringBuilder
注意：StringBuilder没有重写equals()方法，比较字符串时要先转换为String类型
